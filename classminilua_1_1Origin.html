minilua::Origin class

The origin of a value.

Contents

  • Reference
    • Constructors, destructors, conversion operators
    • Public functions

Defaults to NoOrigin.

Using BinaryOrigin and UnaryOrigin you can build a tree to track the changes made to a Value while running a lua program.

You can manually walk the tree using the variant returned by Origin::raw.

Supports equality operators.
